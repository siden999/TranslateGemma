# TranslateGemma v1.1.4

這個版本補上 Windows 第一段安裝的實際指令，避免文件只寫方向卻沒有可操作步驟。

## Windows 第一段安裝

Windows 使用者現在可以開 PowerShell，貼上這一整行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/siden999/TranslateGemma/main/setup/windows/install.ps1 | iex"
```

這一行會自動：

- 下載最新 GitHub Release 的 Windows 安裝包。
- 檢查 Python 3.10-3.12。
- 如果缺 Python，嘗試用 winget 安裝官方 Python 3.12。
- 安裝本機 Launcher、server 依賴與 Native Host。
- 設定開機自動啟動。
- 檢查 `127.0.0.1:18181` 控制服務。

`%LOCALAPPDATA%\TranslateGemma` 只是安裝完成後的位置，不是安裝指令。
