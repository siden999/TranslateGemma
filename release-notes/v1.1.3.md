# TranslateGemma v1.1.3

這個版本主要更新安裝說明與後續開發方向。

## 安裝策略

TranslateGemma 會維持自己的本機模型路線，不把模型服務交給其他工具。

正式流程固定為二段式：

```text
1. 先安裝本機程式
2. 再安裝 Chrome 擴充功能
3. 第一次按啟動時下載模型
```

## Windows

Windows 沒有正式簽章前，未簽章 `.exe` 可能被 Smart App Control 擋住。

接下來 Windows 的短期主流程會改成：

```text
開 PowerShell
貼上 TranslateGemma 提供的一段安裝指令
等本機程式安裝完成
再到 Chrome 載入 extension 資料夾
```

使用者不需要自己操作 venv、pip、PowerShell 腳本或 server。

## macOS

macOS 的短期主流程仍是：

```text
下載 TranslateGemmaInstaller-v版本號.command
右鍵打開
依畫面提示完成安裝
再到 Chrome 載入 extension 資料夾
```

如果缺 Python，安裝器應該清楚提示如何安裝 Python 3.12。
